The Allegra InfoBot

Allegra is the info-bot running on the #Ada channel of the freenode IRC network.

Allegra's source repository is now online! Use git to fetch it: 

git clone git://ftp.niestu.com/allegra

See the file doc/allegra.html in the repository, or online, for tips on building the code. To build Allegra from source, you also need NAIL, a small infrastructure library containing some routines that previously were distributed as part of the main Allegra package.

Two historical pages related to Allegra's development are:

  • The "Hello, world" LP Example
  • The Alba Technology Demonstrator

Allegra was converted from LP to "flat" source during her development, though not without a large measure of regret. There were reasons, and perhaps someday she'll be converted back.
